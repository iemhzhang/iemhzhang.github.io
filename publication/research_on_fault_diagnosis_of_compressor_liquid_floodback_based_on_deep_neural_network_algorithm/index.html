Research on Fault Diagnosis of Compressor Liquid Floodback Based on Deep Neural Network Algorithm

Zhang Menghua, Huazhong University of Science and Technology
July 2020
Cite Project Slides

Abstract

In order to improve the detection rate of compressor liquid return faults in multi-line systems, this paper proposes for the first time a multi-line compressor liquid floodback fault diagnosis model based on deep learning algorithm (Deep Neural Networks, DNN). Fault diagnosis is divided into four main steps ,data variable selection, initial model establishment, DNN model training and fault diagnosis classification prediction. In the experiment, a total of 3 compressor states were set, 17 characteristic variables were selected, and a deep neural network model was established. The results show that the deep learning model can detect the two types of liquid floodback faults more efficiently, with an accuracy rate of 99.86%. Compared with the decision tree model of the supervised algorithm, the correlation analysis and pruning process are not required, and it is compared with the aggregation of the unsupervised algorithm. The class analysis algorithm model does not need correlation and principal component analysis process, the processing process is simple and easy to operate, and the efficiency is increased by 3.48% and 5.91% respectively compared with the two.
